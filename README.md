# YummyDoors Desktop Next.js

Dedicated Next.js frontend for YummyDoors.

## Scope

- customer/restaurant/admin web surface for YummyDoors
- separate from `yummy-desktop-nextjs` POS desktop
- connected to `yummydoors_backend`

## Quick start

1. Copy `.env.example` to `.env.local`
2. Set `NEXT_PUBLIC_API_BASE_URL`
3. Install dependencies
4. Run:

```bash
npm install
npm run dev
```

## Current pages

- `/login`
- `/dashboard`
